## ADDED Requirements

### Requirement: Workbench MUST support OpenSpec change export mode
The workbench MUST let operators switch export from generic filesystem mode to an OpenSpec change target mode.

#### Scenario: Change target mode shown
- **WHEN** a reviewed draft is ready for export
- **THEN** the UI offers an OpenSpec change target mode
- **AND** the operator can choose a change and artifact target from that mode
